Pro

  • Goede mobiele interface  https://www.home-assistant.io/docs/frontend/mobile/
  • Alle instellingen en definities op 1 locatie, zelfs in 1 file
  • Geheel geschreven in Python
  • Zeer grote gebuikersgroep (wel veel techneuten), zeer snel antwoord
  • Interfaced zeer soepel met RFLink, MQTT-broker, http

  • Wat simpele layout. Er is een nieuwe layout in aanbouw: LoveLace die er zeer veelbelovend uitziet.
  • Is voornamelijk bedoeld voor Raspberry, windows installatie lukt daarom soms goed, soms zeer moeizaam
